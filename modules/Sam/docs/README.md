## Introdução
O SAM (Sistema de Atendimento e Monitoramento) é uma plataforma web desenvolvida para gerenciar o fluxo de atendimentos de alunos em instituições de ensino e clínicas multidisciplinares. O sistema integra gestão de alunos, agenda de profissionais, prontuários eletrônicos e auditoria de ações, garantindo conformidade e eficiência no acompanhamento pedagógico e clínico.

## Tecnologias Utilizadas
- **Frontend**: Next.js 16 (App Router), React 19, Tailwind CSS.
- **UI/UX**: Radix UI, Lucide React, Recharts.
- **Backend-as-a-Service**: Supabase (PostgreSQL, Auth, RLS).
- **Linguagem**: TypeScript.
- **Comunicação**: SMTP (Nodemailer), Webhooks.
