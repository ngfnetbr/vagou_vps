# 6. Manual do Usuário

## 6.1 Introdução
Este manual descreve as principais funcionalidades do SAM para profissionais e coordenadores.

## 6.2 Primeiro Acesso
1. Acesse a URL do sistema.
2. Insira seu e-mail e senha fornecidos pelo administrador.
3. Caso tenha esquecido a senha, utilize o link "Esqueci minha senha".

## 6.3 Gestão de Alunos
### Cadastrar Novo Aluno
1. Clique em **Alunos** no menu lateral.
2. Clique no botão **Novo Aluno**.
3. Preencha o nome completo, data de nascimento e selecione a escola/turma.
4. Clique em **Salvar**.
*(Inserir screenshot da tela de cadastro de alunos aqui)*

### Consultar Prontuário
1. Na lista de alunos, clique no nome do aluno desejado.
2. A aba **Prontuário** exibirá todo o histórico de evoluções em ordem cronológica.
*(Inserir screenshot do prontuário do aluno aqui)*

## 6.4 Realizando Atendimentos
### Agendar Sessão
1. Vá em **Agenda**.
2. Clique no dia/horário desejado ou no botão **Novo Agendamento**.
3. Selecione o aluno e o tipo de atendimento.
4. Salve o agendamento.

### Registrar Evolução
1. Durante ou após o atendimento, localize o agendamento na agenda ou lista.
2. Clique em **Realizar Atendimento**.
3. No campo **Evolução**, descreva o progresso do aluno.
4. Clique em **Finalizar**.
*(Inserir screenshot do formulário de atendimento aqui)*

## 6.5 Relatórios
1. Acesse o módulo **Relatórios**.
2. Selecione os filtros desejados (Data, Profissional, Escola).
3. Visualize os dados na tela ou utilize o botão **Imprimir/Exportar**.
