# VAGOU - Apresentação do Sistema

## O que é o VAGOU?

O **VAGOU** é um sistema de gestão de vagas em creches municipais (CMEIs) desenvolvido em plataforma **low-code**, totalmente **web** e **responsivo**.

---

## Características Principais

### 🌐 100% Web
- Acesso via navegador (Chrome, Firefox, Safari, Edge)
- Não requer instalação de software
- Funciona em qualquer dispositivo com internet

### 📱 Responsivo e Mobile
- Interface adaptável para desktop, tablet e celular
- Aplicativo PWA (Progressive Web App) instalável
- Versões nativas Android/iOS disponíveis

### 🔧 Low-Code / No-Code
- Desenvolvido na plataforma **Lovable**
- Configurações via interface administrativa (sem programação)
- Customização de formulários, campos, cores e textos
- Templates de notificação editáveis

### ☁️ Cloud (Nuvem)
- Hospedagem em nuvem (Supabase)
- Backups automáticos
- Escalabilidade automática
- Alta disponibilidade

### 🔒 Seguro
- Autenticação com e-mail/senha ou Google
- Controle de acesso por perfis (RBAC)
- Dados criptografados
- Auditoria completa de ações

---

## Tecnologias Utilizadas

| Camada | Tecnologia |
|--------|------------|
| **Plataforma** | Lovable (Low-Code) |
| **Frontend** | React, TypeScript |
| **Backend** | Supabase (PostgreSQL) |
| **Hospedagem** | Cloud (Supabase/Vercel) |
| **E-mail** | Resend |
| **Integrações** | Webhooks (WhatsApp, SMS) |

---

## Módulos do Sistema

1. **Portal Público** - Inscrições, consulta de fila, ocupação
2. **Área do Responsável** - Acompanhamento, documentos, chat
3. **Área Administrativa** - Gestão completa de vagas e matrículas
4. **Relatórios** - Exportação Excel, CSV, PDF
5. **Configurações** - Personalização completa

## Requisitos Mínimos

**Para usuários:**
- Navegador atualizado (Chrome, Firefox, Safari, Edge)
- Conexão com internet

**Para implantação:**
- Conta Supabase (gratuita ou paga)
- Domínio próprio (opcional)

---

## Implantação

O sistema pode ser implantado em **menos de 1 hora**:

1. Criar projeto no Supabase
2. Executar scripts SQL de setup
3. Configurar autenticação
4. Personalizar dados do município
5. Publicar
